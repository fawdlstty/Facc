# Facc

Facc 是一种自制语言的工具，通过极简语法描述文法，自动生成AST代码。

## 引言

编译原理自始至终都是非常难学的知识，虽然网上能找到各种各样的教程及文档，但也极少有开发者深入研究。本仓库作为另一种方案，以更简单的视角来解读编译原理，提供完善教程协助用户自主完成一个编译器。
